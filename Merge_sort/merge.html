Funkce merge

Pseudokód funkce Merge

array: pole prvů
n: délka pole


merge (array, left, middle, right)
	sizeLeft = middle - left + 1
	sizeRight = right - middle

	vytvor leftArray delky sizeLeft
	vytvor rightArray delky sizeRight

	//kopiruju prvky z array do leftArray
	for i = 0 to sizeLeft
		leftArray[i] = array[left+i]
	end for

	//kopiruju prvky z array do rightArray
	for i = 0 to sizeRight
		rightArray[i] = array[middle + i + 1]
	end for

	leftIndex = 0
	rightIndex = 0

	arrayIndex = left

	//porovnávám prvky v obou polích a menší vložím do pole array
	while(leftIndex < sizeLeft and rightIndex < sizeRight)
		if leftArray[leftIndex] <= rightArray[rightIndex]
			array[arrayIndex] = leftArray[leftIndex]
			leftIndex++
		end if
		else
			array[arrayIndex] = rightArray[rightIndex]
			rightIndex++
		end else
		arrayIndex++
	end while

	//dokopíruju zbytek pole leftArray (pokud jsou zde ještě nějaké prvky)
	while leftIndex < sizeLeft
		array[arrayIndex] = leftArray[leftIndex]
		leftIndex++
		arrayIndex++
	end while

	//dokopíruju zbytek pole rightArray (pokud jsou zde ještě nějaké prvky)
	while rightIndex < sizeRight
		array[arrayIndex] = rightArray[rightIndex]
		rightIndex++
		arrayIndex++
	end while

Merge sort

Pseudokód funkce Merge sort

array: pole prvků
left: zacatek pole
right: konec pole

mergeSort(array, left, right)
	if left < right
		middle = (left + right)/2
		mergeSort(array, left, middle)
		mergeSort(array,middle+1, right)

		merge(array, left, middle, right)
	end if
