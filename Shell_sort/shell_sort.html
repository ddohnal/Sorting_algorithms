Shell sort

Shell sort je zobecněná verze algoritmu Insertion sort. Nejprve seřadí prvky, které jsou od sebe vzdálené, a postupně zmenšuje interval mezi řadícími prvky.

Interval mezi prvky se zmenšuje na základě použité posloupnosti. Některé z optimálních posloupností, které lze použít v algoritmu shell sort, jsou:

  1. Původní Shellova posloupnost: $ \lfloor{ \dfrac{n}{2^k}} \rfloor \rightarrow \dfrac{n}{2}, \dfrac{n}{4}, \dots , 1$.
  2. Hibbardova posloupnost: $2^k-1 \rightarrow 65535, 32767, 16383, 8191, 4095, 2047, 1023, 511, 255, 127, 63, 31, 15, 7, 3, 1$.
  3. Stasevichova posloupnst: $2^k+1 \rightarrow 65537, 32769, 16385, 8193, 4097, 2049, 1025, 513, 257, 129, 65, 33, 17, 9, 5, 3, 2, 1 $.
  4. Ciurova posloupnost: empiricky $ \rightarrow 701, 301, 132, 57, 23, 10, 4, 1$.

Ukázka

V ukázce je použita původní Shellova posloupnost.

Pseudokód

Shell sort

array: pole prvů
n: délka pole

for interval = n/2 to 0; interval /= 2
	for i = interval to n
	t = array[i]
	j
		for  j = i; j >= interval and array[j-interval] > temp; j -= interval
			array[j]=array[j-interval]
		end for
	array[j] = t
end for

Složitost

Shellovo třídění vychází z Insertion sortu ($O(n^2)$). Aplikujme nyní třídění Insertion sortu ne na celou posloupnost ale na $h$ jejich podposloupností ($h>1$).Každá podposloupnost bude obsahovat řádově $\dfrac{n}{h}$ prvků. Časová složitost u každé takovéto podposloupnosti je $\dfrac{(\dfrac{n}{h})^2}{4} = \dfrac{n^2}{4h^2}$.Těchto posloupností je však $h$, celkově tedy $h\cdot \dfrac{n^2}{4h^2} = \dfrac{n^2}{4h}$. Což je výrazně méně, než u klasického Insertion sortu. Výsledná časová složitost Shell sortu je výrazně závislá na vybrané volbě mezer. Nejlepší výsledek, jaký můžeme dosáhnout je $O(n^{\frac{4}{3}})$, obecně však $O(n^2)$.
